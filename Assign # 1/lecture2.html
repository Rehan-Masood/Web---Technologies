Lecture 2: HTML Basics

🖥️ Overview

HyperText Markup Language (HTML) is the foundation of all web pages. This lecture covers the fundamental HTML elements, structure, and semantics needed to create well-formed, accessible web documents. We will explore how HTML documents are structured and the purpose of key tags in creating meaningful web content.

🎯 Key Topics

  • HTML Document Structure: DOCTYPE, html, head, and body tags
  • Semantic HTML: Using meaningful tags for better accessibility and SEO
  • Text Formatting: Headings, paragraphs, emphasis, and strong tags
  • Lists and Links: Creating ordered, unordered, and definition lists
  • Images and Multimedia: Embedding images and media in web pages
  • Attributes: Understanding and using HTML attributes effectively

✅ Learning Outcomes

After completing this lecture, you will be able to:

  • Write valid HTML5 documents with proper structure
  • Use semantic HTML tags appropriately
  • Create lists, links, and embed media elements
  • Understand HTML metadata and document meta tags
